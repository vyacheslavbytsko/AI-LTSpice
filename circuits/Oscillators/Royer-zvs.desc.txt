Схема Spice, представленная в файле `Royer-zvs.net`, является схемой генератора осциллятора Royer с функцией ZVS (Zero Voltage Switching), который представляет собой тип высокочастотного генератора, использующий две пара транзисторов и диодов для создания высокочастотного сигнала.

Эта схема состоит из следующих элементов:

* **Транзисторы (M1 и M2)**: Два транзистора IRFZ44N, используемые в качестве переключателей для создания высокочастотного сигнала.
* **Диоды (D1 и D2)**: Два диода, используемые для ограничения напряжения и создания правильного режима работы транзисторов.
* **Катушки индуктивности (L1, L2 и L3)**: Три катушки индуктивности, используемые для создания колебательного контура и фильтрации высокочастотных гармоник.
* **Конденсатор (C1)**: Один конденсатор емкостью 330 нФ, используемый для создания колебательного контура.
* **Резисторы (R1, R2 и R3)**: Три резистора, используемых для ограничения тока и создания правильного режима работы схемы.
* **Источник питания (V1)**: Один источник питания постоянного тока с напряжением 15 В.

Эта схема работает следующим образом:

1. Транзисторы M1 и M2 включаются и выключаются с высокой частотой, создавая переменный ток в катушках индуктивности L1 и L2.
2. Колебательный контур, состоящий из катушек индуктивности L1 и L2, конденсатора C1 и диодов D1 и D2, создает высокочастотный сигнал.
3. Резистор R2 ограничивает ток через транзисторы, предотвращая их перегрев и повреждение.
4. Катушка индуктивности L3 используется для фильтрации высокочастотных гармоник и предотвращения скачков напряжения на источнике питания.

Схема осциллятора Royer с ZVS является популярным выбором для высокочастотных приложений, таких как генерация импульсов, радиочастотная энергетика и другие.
